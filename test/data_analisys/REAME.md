# Data Analisys

This folder contains some of the files used to test FlowScanner and other tools deteciton capabilies, raw data in the "data" folder, processed data in the "data/processed_data" folder. The csv format is based on an internal tool we used for the tests.

Data used for the "/usr/bin" memory usage statistics in the "data/processed_data" folder, in particular, the "memory_usage.csv" file contains all pages accesed for execution by each program with the virtual address and number of bytes used, while the "memory_summary.csv", contains the average and max bytes per page used by each application.

The python script can be used to read or process the raw data.
